# 快捷鍵備忘
- alt 加上下鍵
- Ctrl + U 回到上一個滑鼠位置
- Ctrl + PageUp 切換tab
- Ctrl + Shift + PageUp 移動Tab
- F2 快速更改 該變數名稱
